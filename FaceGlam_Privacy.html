1. Information We Do Not Collect


We want to assure you that we do not collect, store, or share any personal data from our users. This includes, but is not limited to:

Usage data or analytics.

Photographic or video data.

2. Use of TrueDepth API


Our app utilizes Apple's TrueDepth API to provide a more accurate and immersive makeup experience. Here’s how we use this technology:


Face Mapping: The TrueDepth API helps us to map your facial features accurately so that the makeup application looks as realistic as possible.

Real-Time Processing: All the processing is done in real-time on your device. None of the data captured by the TrueDepth API is sent to our servers or any third party.

Data Security: The facial data used by the TrueDepth API stays on your device and is never stored or transmitted.

3. No Data Storage


Since we do not collect any data, there is no storage of personal or non-personal data. All processes occur locally on your device.
